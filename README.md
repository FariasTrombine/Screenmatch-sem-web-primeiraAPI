ScreenMatch

Projeto desenvolvido durante os estudos da plataforma Alura com o objetivo de praticar o desenvolvimento de APIs em Java utilizando Spring Boot.

Sobre o projeto:
Ate o momento, a aplicação consulta serie por meio de consumo de API externa, permitindo obter informações como titulo, total de temporada, episódios, ano de lançamento
O projeto foi criado com foco no aprendizado de conceitos fundamentais no mundo do backend, como consumo de API's, listas, manipulação de dados Json.

Tecnologias Utilizadas
Java
Spring Boot
Maven
Jackson
API REST
Git e GitHub

Conceitos Praticados
Programação Orientada a Objetos (POO)
Consumo de APIs REST
Serialização e desserialização de JSON
Organização em camadas
Tratamento de dados recebidos de serviços externos
